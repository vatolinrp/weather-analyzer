ЗАДАНИЕ НА ПРЕДДИПЛОМНУЮ ПРАКТИКУИзучить существующие погодные веб-службы, выяснить спектр метеоданных, доступных в бесплатном режиме для разработчика, изучить методы получения и модели хранения данных, выбрать веб-службы и интегрировать их работу с технологией Apache Storm, которую заранее следует изучить. Изучить работу с кэшем Ehcache, для хранения текущих данных. Изучить JavaMail API, для отправки отчетов на электронную почту ответственному лицу. Изучить и интегрировать создание отчета в формате CSV.
